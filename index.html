A setting for use in playing TTRPG games. Specifically, in games run with Redblast1 :P . The button below will walk you through making a character in this world. Included are instructions on how to create a character in the Pathfinder 2E system, too!

Get Started

What's Mergi?

Mergi is a high fantasy custom setting made by Redblast1. It is a continent found trapped within tall scorching-hot volcanic rock walls. Tension has started to build once again between the factions around Mergi: the constant fighitng of the Monarchy with the Indipendent City-States, or the Neutral factions facing a looming threat from the sea, sky, or the mists. However, one thing is clear: the Quest for Ascension continues. Who will find a way to clear the Cage?

Easy To Make A Character In!

Just work with Red and he'll work with you and figure out some way to make it work- workshop how your crazy character will work in this crazy world. The instructions will help you walk-through the steps.

No Ancestry Or Class Restrictions!

The world doesnt have restrictions on what to select- if it doesnt already have roots in the world, we can figure that out.
